# CS50 Python Projects

A collection of problem sets completed in Python as part of Harvard's CS50: Introduction to Computer Science. These projects revisit core programming logic from the C section of the course, translated into Python, alongside new algorithmic challenges.

## Projects

### DNA
Identifies a person from a small DNA database by counting the longest run of consecutive repeats (STRs) in their DNA sequence and matching it against known profiles in a CSV file.

**Run:**
```bash
python dna.py database person-dna
```

### Credit
Python implementation of a credit card validator using Luhn's algorithm — a direct comparison to the C version, applying the same logic with Python's simpler syntax and built-in data structures.

**Run:**
```bash
python credit.py
```

### Cash
Calculates the minimum number of coins for change using a greedy algorithm, rebuilt in Python with input validation.

**Run:**
```bash
python cash.py
```

### Readability
Estimates the reading grade level of a passage of text using the Coleman-Liau index.

**Run:**
```bash
python readability.py
```

## Requirements
- Python 3.x

## About
These projects were rebuilt in Python after first solving them in C, as part of strengthening core problem-solving and algorithmic thinking across different languages and paradigms.
